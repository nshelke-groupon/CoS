# Integrations

## Overview

Backbeat has three external dependencies, all outbound: the Accounting Service callback endpoint (activity/decision notifications), the Metrics Stack (operational telemetry), and the SMTP Relay (daily digest emails). There are no inbound federated integrations with other Continuum internal services documented in the architecture model — upstream consumers interact with Backbeat through its REST API.

## External Dependencies

| System | Protocol | Purpose | Critical | Architecture Ref |
|--------|----------|---------|----------|-----------------|
| Accounting Service (callback endpoint) | REST / HTTPS | Receives activity and decision callbacks from Backbeat after workflow event execution | yes | `accountingServiceEndpoint` (stub-only) |
| Metrics Stack | InfluxDB/Sonoma | Receives operational metrics and error telemetry from Worker Runtime | no | `metricsStack` |
| SMTP Relay | SMTP | Delivers daily workflow activity reports generated by the Daily Activity Reporter | no | `smtpRelayEndpoint` (stub-only) |
