## Troubleshooting

### UI Shows "Loading..." Indefinitely

- **Symptoms**: Browser shows `Loading...` text; React app does not mount
- **Cause**: The SPA bundle failed to load, or `/api/mbus-sigint-frontend/app-config` or `/api/mbus-sigint-frontend/session-info` failed to respond
- **Resolution**: Check ELK logs for Node.js errors; verify the app-config and session-info endpoints return 200 on the Hybrid Boundary URL; check Okta/HB authentication headers are being forwarded

### Configuration Change Request Submission Fails

- **Symptoms**: User sees error when submitting a new change request via the UI form
- **Cause**: `mbus-sigint-config` POST `/change-request` endpoint is returning an error, or CSRF token mismatch
- **Resolution**: Check `mbus-sigint-config` health; verify session-info CSRF token is correctly included in request; check ELK logs for upstream 4xx/5xx responses

### Service Names Autocomplete Not Working

- **Symptoms**: The service name autosuggest field in the request form shows no suggestions
- **Cause**: The `/api/service-portal/services.json` proxy call is failing
- **Resolution**: Check `service-portal` health at `http://service-portal.production.service`; the SPA will retry automatically every 5 seconds

### Pod Crashloop

- **Symptoms**: Kubernetes reports pod in CrashLoopBackOff
- **Cause**: Node.js startup failure — likely misconfigured environment variable or missing `keldor-config` stage file
- **Resolution**: Check pod logs via Kube Web View or napistrano: `npx nap --cloud logs --follow production us-central1`
