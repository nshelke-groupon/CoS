# Events

## Overview

This service does not publish or consume async events. `clo-ita` is a synchronous request/response BFF. All interactions with downstream services are performed via synchronous HTTPS/JSON calls initiated by incoming HTTP requests. There is no integration with Kafka, RabbitMQ, SQS, or any other messaging system.
