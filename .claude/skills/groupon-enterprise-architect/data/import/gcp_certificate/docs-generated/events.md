# Events

## Overview

This service does not publish or consume async events. The `gcp_certificate` service is a Terraform/Terragrunt IaC repository that manages GCP Private CA infrastructure. There is no runtime process, message broker integration, or event-driven behavior. Certificate issuance by downstream consumers occurs synchronously via direct GCP Private CA API calls authenticated with IAM permissions provisioned by this service.
