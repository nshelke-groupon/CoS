# Marketing Deal Service

## Purpose
Marketing Deal Service manages merchant deal creation, feed generation, and performance reporting.

## Scope
- Exposes merchant APIs for deal setup.
- Generates partner feeds consumed by Encore Platform.
- Produces performance metrics for internal dashboards.

## Tech Stack
- Java 17
- Spring Boot 3
- PostgreSQL
- Kafka

## Status
- Modeled under Continuum (federated import) with dedicated database and relationships.
