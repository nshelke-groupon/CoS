## Overview

AIDG exposes two internal Java-based REST APIs within the Continuum platform. The InferPDS API provides enrichment endpoints for PDS (Product Data Service) inference, and the Merchant Quality API provides endpoints for retrieving merchant quality scores. Both are internal services; they are not directly internet-facing and are consumed by other Continuum platform services.
