# Events

## Overview

This service does not publish or consume async events via a message broker. The Cloudability Metrics Agent pushes metric samples directly to Cloudability's HTTPS ingestion endpoint on a periodic schedule managed internally by the agent runtime. There is no Kafka, RabbitMQ, SQS, or Pub/Sub integration.
