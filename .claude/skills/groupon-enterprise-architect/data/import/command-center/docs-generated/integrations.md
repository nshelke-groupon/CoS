# Integrations

## Overview

Command Center integrates with five internal Continuum platform services and three external systems. The web container (`continuumCommandCenterWeb`) calls downstream APIs synchronously to read current state before scheduling operations. The worker container (`continuumCommandCenterWorker`) calls downstream APIs asynchronously as part of batch job execution. All HTTP integrations use JSON over HTTP. Salesforce integration uses REST. Message bus integration uses the internal MBus protocol.
